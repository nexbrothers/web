## What students actually need from a scanner

Before comparing apps, let's agree on the criteria. A student scanner should:

- **Be free** — students don't have $50/year for a subscription
- **No watermark** — submitting a watermarked PDF to a professor is unprofessional
- **Work offline** — campus Wi-Fi is unreliable; airplane mode in lectures
- **Handle handwritten notes well** — most filters ruin pencil and ink
- **Support multi-page batches** — for scanning textbook chapters
- **Have OCR** — to make notes searchable
- **Be fast** — between classes, you have 5 minutes max
- **Respect privacy** — your ID cards and exam forms shouldn't end up on a server

These are the criteria I'm going to score every app on.

## Common student scanning workflows

Here are the situations where a student typically reaches for a scanner:

### Scanning lecture notes
You write longhand notes in class, then scan them at the end of the day. Use the Sharpen filter (Scanvo) or default (Lens) for handwriting. Add to a per-subject PDF using Add Page.

### Scanning textbook chapters before an exam
Borrow a friend's textbook, scan the chapter you need, return it. Use the Best filter for printed text. 50+ pages = use a phone tripod and a stable surface.

### Scanning student ID for an online portal
Most college portals require a JPG or PDF of your student ID. Use Scanvo's ID Card mode to capture it cleanly with no glare.

### Submitting an assignment as PDF
Scan handwritten or printed pages, combine into one PDF with Add Page, run OCR if your professor needs to copy text, export and email.

### Filing for hostel forms / fee receipts
Use the Receipts approach — scan, label with date, OCR for searchability, store in a year-by-year folder.

### Scanning your passport for a visa or program application
Different from ID card mode — passports need Scan Document mode for the larger format. See our [passport scanning guide](/blog/how-to-scan-passport-for-visa-application/).
