You have 3 PDFs that need to be combined into one — maybe scanned receipts for an expense report, or chapters of a study guide, or a multi-part contract. Most "merge PDF" websites either limit you to 2 files in the free tier, watermark the output, or require you to upload your documents to a server you don't trust.

Here's how to merge any number of PDFs on your Android phone, offline, in under a minute.

## Quick answer

Open **[Scanvo](https://play.google.com/store/apps/details?id=com.nexbrothers.scanvo)**, tap **Tools → Merge PDF**, select all the PDF files you want to combine, drag to reorder them, and tap **Merge**. You get one PDF with all the pages in order.

## Why offline merging matters

Online PDF mergers all have at least one of these problems:

- **File size limits**: Most are capped at 5-25 MB total
- **File count limits**: Free tier usually allows 2-3 files
- **Watermarks**: "Merged with FreeTool.com" splash on every page
- **Privacy**: Your PDFs sit on a server you don't control
- **Time**: Upload + process + download takes 30-90 seconds per merge
- **Internet required**: Useless when you need it most (in a meeting, at the airport)

A native phone app fixes all of this: instant, free, no limits, no privacy compromise.

## Step-by-step guide

### Step 1: Install Scanvo
Get **Scanvo** from the Play Store. It's a complete PDF toolkit — scan, merge, sign, OCR, compress — all free, all offline.

[Download Scanvo on Google Play](https://play.google.com/store/apps/details?id=com.nexbrothers.scanvo)

### Step 2: Open the Merge PDF tool
From the Scanvo home screen, tap **Tools**. You'll see a grid of PDF utilities. Tap **Merge PDF**.

### Step 3: Select the PDF files
Tap **Select PDFs**. Your phone's file picker opens. Navigate to the folder containing your PDFs and tap each one to select. You can pick PDFs from anywhere — Downloads, WhatsApp, Drive (if synced locally), or Scanvo's own document library.

There's no limit on how many you can pick. We've tested with 50+ files.

### Step 4: Reorder the merge sequence
Scanvo shows you a list of selected PDFs with thumbnails. The merge order is the order shown — top to bottom.

To reorder:
- Long-press a row and drag it to a new position
- Tap the trash icon next to any file to remove it from the merge

Make sure the order is what you want before continuing — once merged, you can't easily reorder pages without re-merging.

### Step 5: Set merge options
Tap the gear icon for advanced options:

- **Add page numbers**: Adds a small page number to the bottom-right of each page
- **Add bookmarks**: Each source PDF becomes a bookmark in the merged file (useful for navigation in long merges)
- **Compress output**: Reduces the merged PDF size (slight quality trade-off)

For most use cases the defaults are fine.

### Step 6: Tap Merge
Scanvo combines the PDFs in the order you specified. For typical cases (5-10 PDFs), this takes 1-2 seconds. Larger merges may take longer.

### Step 7: Save or share
Tap **Save** to keep the merged PDF in your Scanvo library, or **Share** to send it via email, WhatsApp, or any other app. You can also tap **Open File** to view it in your default PDF reader.

## Pro tips

- **Compress before merging** if your input PDFs are very large. Use **Tools → Compress PDF** on each file first. The merged result is much smaller.
- **Add bookmarks** for long merged documents. When you open the merged file in a PDF reader, you can jump between sections instantly.
- **Don't merge password-protected PDFs**. Remove the password first using Tools → Unlock PDF (if you know the password). Otherwise the merge will fail.
- **For consistent page sizing**, all source PDFs should be the same page size. Mixing A4 and Letter creates a slightly inconsistent merged file — readable, but not ideal.
- **For scanned documents**, use Scanvo's **Scan Document** mode to scan everything fresh into a single multi-page document instead of scanning then merging. Faster overall.

## Frequently asked questions

**How many PDFs can I merge at once?**
There's no hard limit. Tested up to 50 files. Larger numbers are theoretically supported, limited only by your phone's RAM.

**Is there a maximum file size?**
No software limit. Practically, your phone's storage and RAM are the only limits. We've successfully merged 100+ MB outputs.

**Will the merge add a watermark?**
No. Scanvo never adds watermarks on any tier.

**Can I merge PDFs of different page sizes?**
Yes. The merged PDF preserves each source's original page size. Pages will appear at their natural dimensions, which can look slightly inconsistent — but it's perfectly readable.

**Can I split a merged PDF later?**
Yes. Use Tools → Split PDF to break a merged document back into separate files by page range or by bookmarks.

**Are my PDFs uploaded anywhere during the merge?**
No. The entire merge happens on your device. No internet is required, and nothing is sent to any server.

**Does the merged PDF lose quality?**
No, unless you enable compression. The default merge preserves 100% of the original content.

**Can I add a cover page or table of contents?**
For a cover page, scan or create a title page first, then include it as the first item in the merge. For a table of contents, use the bookmarks feature — each source PDF becomes a navigation entry.
